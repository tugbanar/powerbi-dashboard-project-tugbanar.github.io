📊 Power BI Projesi – Spotify ilk 200 Albüm Analizi

##  Proje Açıklaması
Bu projede Spotify üzerinden elde edilen 20-26 mart tarihleri arasındaki dinlenme verileri analiz edilmiştir. 
Veriler Power BI kullanılarak görselleştirilmiş ve trend analizleri yapılmıştır.

##  Kullanılan Araçlar
- Power BI
- Veri toplama (Spotify verisi)
- Veri görselleştirme

##  İçerik
- .pbix analiz dosyası
- Dashboard görselleri

##  Amaç
Müzik dinlenme alışkanlıklarını analiz ederek trendleri ve kullanıcı davranışlarını incelemek.
